CellMag Magnetic Base / PowerMag High-Efficiency Magnetic Column

This page focuses on CellMag magnetic bases and PowerMag high-efficiency magnetic columns. Both can be used in magnetic bead cell sorting systems, but their roles are different: CellMag undertakes the magnetic separation operation platform, while PowerMag is used to improve the efficiency of magnetic bead sorting, and is suitable for planning together with consumables and automatic detection systems.

CellMag

Magnetic separation operating platform, suitable for 12 x 75 mm test tubes and magnetic bead sorting applications

  • Can be used in magnetic bead cell sorting systems of various factories.
  • There is a high magnetic field inside to separate cells labeled with magnetic beads.
  • supports positive or negative screening processes.
  • Designed for use with 12 x 75 mm (5 mL) polystyrene tubes and PowerMag magnetic columns.
PowerMag

uses magnetic columns to enhance magnetic bead sorting efficiency, and is suitable as a process efficiency component.

  • can also be used in magnetic bead cell sorting systems of various manufacturers.
  • The cell sorting efficiency using magnetic columns is more than five times higher than that without magnetic columns.
  • Suitable for use with reagent/consumable sets and magnetic holders.
  • It can be used as an important synergistic component in leukocyte removal and magnetic bead sorting processes.

Safety reminder

  • People wearing heart rate regulators need to keep at least 30 centimeters away from the magnetic base.
  • Electronic products with magnetic fields should be kept away from the magnetic base to avoid demagnetization.
  • When planning usage scenarios, it is necessary to reserve a safe operating space for magnetic products.

FAQ

When would you need to use both?

PowerMag is a magnetic column component that improves sorting efficiency. It usually needs to be planned together with the magnetic base and the consumable process to truly maximize the value of the process.
CellMag can be used on magnetic bead cell sorting platforms, while PowerMag is a magnetic column that enhances sorting efficiency. Whether to use them at the same time should be confirmed based on the actual process and product configuration.
