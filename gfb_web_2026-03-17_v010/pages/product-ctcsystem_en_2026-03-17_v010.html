Chiline CATCH Automatic detection system

Chiline CATCH circulating tumor cells automatic detection system is a set of fully automated detection equipment that can be connected in series with procedures such as blood sample processing, circulating tumor cells separation and staining, image analysis and report output, allowing technicians to perform the CTC detection process in a more standardized manner.

Process description

Integrate blood sample processing, separation, staining and image analysis into one process

  • After placing blood samples, detection reagents and consumables, the automatic program can be started.
  • The system will complete blood sample processing, circulating tumor cells separation and immunofluorescence staining in sequence.
  • Microscope photography, image analysis and report output are then automatically performed.
  • Finally, the image analysis and report data are integrated as the basis for sorting out the test results.
